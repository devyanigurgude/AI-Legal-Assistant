# Frontend (React + Vite)

This is the web dashboard for AI Legal Assistant.

## Requirements
- Node.js 18+
- npm

## Setup
```bash
npm install
npm run dev
```

The app runs at `http://localhost:5173` by default.

## API Base URL
The frontend currently uses a hardcoded API base URL:
- `frontend/src/services/apiService.ts`

If you want to point to a different backend, update that file or add env‑based support later.
